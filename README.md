# AI Tutor Learning Notes

## 1. Frontend = What user sees
File: app/page.js
- useState: stores data like input text
- onClick: runs code when button clicked
- fetch: sends data to backend

## 2. Backend = The brain
File: app/api/tutor/route.js
- POST: receives data from frontend
- OpenAI API: sends question to ChatGPT and gets answer
- process.env: stores secret keys safely
